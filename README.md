NCSUN Tools Suite
A collection of community intelligence and data tools built by NCSUN CIC to support the Grassroots Intelligence Network Engine (GINE) and associated programmes.
Live site: https://ncsun-cic.netlify.app
Tools included:

GINE Services Database (Station 6) — Norwich, Norfolk & Suffolk services directory with live search and community add function
NLEH Database — Norwich Lived Experience Hub triage and case logging
Prevention Engine — trauma-informed NLEH triage prototype
GINE Master Log — anonymised community intelligence log
GINE Financial Log — financial pilot data
